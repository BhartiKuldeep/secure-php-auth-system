# Contributing

Thanks for contributing.

## Local setup
1. Copy `.env.example` to `.env`.
2. Run `php setup.php`.
3. Start the server with `php -S localhost:8000 -t public`.

## Pull requests
- Keep changes focused and small.
- Explain the reason for the change.
- Run the lint command before opening a PR.
- Update docs when behavior changes.

## Commit style
Use short, clear commit messages, for example:
- `feat: add session activity log`
- `fix: handle expired reset tokens`
